# FNP-Sales-Analysis-
This project analyzes sales data from Ferns and Petals , a gifting company specializing in occasion-based products such as Diwali, Birthdays, and Anniversaries. An interactive dashboard was built to track key business metrics including total revenue, average customer spending, delivery time, monthly sales trends, top products and much more.
This project analyzes sales data from Ferns and Petals (FNP), a leading gifting company in India. The interactive dashboard provides comprehensive insights into sales performance, customer behavior, and product trends across various occasions like Diwali, Valentine's Day, Raksha Bandhan, and more.
